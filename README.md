# Atividade 8 - Listagem de Exercícios JavaScript

Resolução dos 5 exercícios práticos da Atividade 8, aplicando conceitos fundamentais de JavaScript.

## Exercícios

| Arquivo | Descrição |
|---|---|
| `exercicio1.js` | Sistema de Cadastro de Produtos |
| `exercicio2.js` | Sistema de Matrícula Acadêmica |
| `exercicio3.js` | Sistema de Biblioteca |
| `exercicio4.js` | Sistema de Controle Financeiro |
| `exercicio5.js` | Sistema Completo de Eventos |

## Como executar

```bash
node exercicio1.js
node exercicio2.js
node exercicio3.js
node exercicio4.js
node exercicio5.js
```

> Requisito: ter o [Node.js](https://nodejs.org) instalado.
